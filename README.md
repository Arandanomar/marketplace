## Цель проекта:
Оценка продуктовых метрик маркетплейса с целью увеличения выручки. 

## Данные:
1. CUSTOMERS - информация о клиентах
2. ORDERS - информация о заказах
3. ITEMS - информация о товарах

## Результаты исследования:
1. Оценка месячного retention в оформление заказа с помощью когортного анализа.
2. Выяснено, что product/market fit у маркетплейса отсутствует. Какие причины могут быть у этого быть и что можно сделать.
3. Определены 5 основных метрик, на которых продакт может сконцентрироваться, чтобы максимизировать прибыль компании (GMV, MAU, NAU, RR, ARPPU).
4. Для А/B тестирования выбрана гипотеза при помощи фреймворка ICE.
5. Сформулированы метрики, на которые гипотеза должна повлиять.
6. Подготовлен отчет и рекомендации.
